This is a Connect IQ datafield which connects to Cylypso Ultrasonic sensors through BLE. It is showing
- wind rose
- wind direction (TWD or AWD)
- wind speed (TWS or AWS)
- SOG (kt)
- VMG (kt)
- tacking assistent
- roll (if supported by the sensor)

The update frequency can be configured to 4Hz, if supported by the sensor.
